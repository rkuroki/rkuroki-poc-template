# Create PageUrl Model

**Objective:** Implement data access operations for the `PageUrl` entity.

**Steps:**
1. Create `db/pageurl.model.ts`.
2. Define TypeScript types and Zod schemas for `PageUrl`.
3. Implement `getPageUrls`, querying all public page urls.
4. Implement `getPageUrlById`.
5. Implement `createPageUrl`.
6. Implement `updatePageUrl` and `deletePageUrl` (to be used later by the backoffice admin interface).
