# Setup Next.js Base

**Objective:** Initialize the Next.js application, configure TailwindCSS, shadcn/ui, and establish the base directory skeleton.

**Steps:**
1. Initialize a Next.js project using App Router, TypeScript, and the standard TailwindCSS configuration.
2. Review configuration to ensure the application compiles/runs as a single deployable service.
3. Integrate styling prerequisites for shadcn/ui (`components.json`).
4. Establish project directory structure accurately as planned (`app/`, `components/`, `lib/`, `e2e/`, `db/`).
